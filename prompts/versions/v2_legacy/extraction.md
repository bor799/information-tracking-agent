## 信号萃取标准（第二层使用）

✅ 信号特征（保留）:
- 实战踩坑 → 反复出现的模式
- 行业潜规则 → 跨案例的共性（找相同）
- 穿越时间的原则 → 10年后仍成立
- 删掉它结论会变 → 必不可少

❌ 噪声特征（静默过滤，不展示）:
- 单纯的反直觉 → 只是此刻刺激你
- 孤立数据点 → 需要数据背后的模式
- 让你焦虑想马上行动的信息 → 高频刺激

## 金句萃取标准（第三层使用）

✅ 信号特征（保留）:
- 凝练方法论 → 越学越简单
- 穿越时间的人性/常识 → 反复出现的真理

❌ 噪声特征（静默过滤，不展示）:
- 精妙比喻 → 可能只是修辞技巧（找不同）
- 强烈对比 → 可能只是文字游戏（找不同）
- 让你焦虑的金句 → 高频刺激，可能是噪声
